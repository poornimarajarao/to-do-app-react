# To-Do App (React + Vite)

A simple **To-Do App** built with **React** and **Vite** for fast development and modern bundling. This project allows users to add, edit, delete, and mark tasks as complete.

## About
This app includes the following functionalities:
- Add tasks to the list
- Edit existing tasks
- Delete tasks
- Mark tasks as completed with a checkbox 

This project is mainly for **learning purposes** to understand React, state management, and component-based design.

## Features
- Fully responsive layout using Bootstrap
- Dynamic task management (add, edit, delete, toggle complete)
- Input validation to prevent empty tasks
- Clear visual feedback for completed tasks (strikethrough style)
- Modular component structure: `Navbar`, `FormList`, and main `App`

## Hosted on Vercel:

[https://to-do-app-react-nine-mocha.vercel.app/](https://to-do-app-react-nine-mocha.vercel.app/)


## Usage
- Enter a task in the input box and click "Add task" to add it to the list.
- Click the pencil icon to edit a task.
- Click the trash icon to delete a task.
- Check the box to mark a task as completed or uncompleted.

## Installation
1. Clone the repository:
```bash
git clone https://github.com/poornimarajarao/to-do-app-react
```
2. Navigate to the project directory:
```bash
cd to-do-app-react
```
3. Install dependencies:
```bash
npm install
```
4. Start the development server:
```bash
npm run dev
```
5. Open your browser and go to:
```
http://localhost:5173
```
